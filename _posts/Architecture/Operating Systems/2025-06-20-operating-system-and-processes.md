# üöÄ Operating System and Processes
An **Operating System (OS)** is the most essential **program** running on a computer. It manages **hardware resources** and provides an interface for **users and applications** to interact with the underlying system.

> It acts as the **middle layer** between user applications and hardware.

![Three-Layer Architecture of a Computer System](/assets/img/posts/Architecture/OperatingSystems/ThreeLayerArchitectureAfAComputer%20System.jpeg)


## ‚öôÔ∏è Core Responsibilities of a Modern OS

| Function           | Description                                                                 |
|--------------------|-----------------------------------------------------------------------------|
| **Process Management** | Creates, schedules, and terminates processes. Manages CPU allocation.     |
| **CPU Scheduling**     | Determines which process gets CPU time and when.                        |
| **Memory Management**  | Allocates and recycles memory used by programs.                         |
| **Storage Management** | Manages read/write access to disk and file systems.                     |
| **I/O Management**     | Handles input/output devices (e.g., keyboard, mouse, network, display). |
